## I/K-Bus Tool  
LIN-Bus (BMW I/K-Bus Protocol) Logger & Writer  

![Screenshot](https://pub.files.multilan.de/ik-bus-tool/screenshot1.png)
  
### Features  
- Logging  
- Writing frames
- Multiple Formats for Saving (Binary, HEX, Formatted Text)
- Text to Data Converter
- Charset Picker
- Bus Scanner
- Device Emulations (IKE & MFL, experimental!)
  
### Requirements
Requirements are included in the packaged versions.
- PySide6
- pyserial
- python3
- (pip3)
